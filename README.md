# Currency Converter GUI (INR to Foreign)

This is a Java Swing application that allows users to convert Indian Rupees (INR) to various foreign currencies including:

- USD (United States Dollar)
- EUR (Euro)
- JPY (Japanese Yen)
- GBP (British Pound)
